CONTEXT:

PROJECT: La fabrique à poster

OBJECTIVE:
- Create an e-commerce site to sell landscape posters in sketch style.
- Users can upload their photo or choose from a gallery of models.
- The final image is manually transformed by the admin team and sent to the user via email.

MAIN FEATURES:
1. Upload image or select from gallery
2. Add text with font selection
3. Create user account and login
4. Track orders with statuses:
   - Pending transformation
   - In progress
   - Sent for validation
   - Validated
   - Printing
   - Shipped/Delivered
5. Receive final image by email
6. Payment via Stripe or PayPal

TECH STACK:
- Frontend: Next.js + React
- Backend/BaaS: Supabase (Auth, PostgreSQL DB, Storage, Functions)
- Storage: Secure storage for uploaded and final images
- Emails: SendGrid/Mailgun via Supabase Functions
- Hosting: Vercel recommended (Next.js), or Cloudflare Pages for frontend static parts
- Domain: OVH, DNS via Cloudflare

CONSTRAINTS:
- Images in gallery: low-res + watermark, right-click disabled
- Backend: secure with HTTPS
- Emails: images sent securely for validation
- Payment: PCI-DSS compliant
- No automatic AI generation on the site
- Focus on usability, secure image handling, and manual workflow

USER FLOW:
1. User visits site via domain
2. Chooses an image or uploads a photo
3. Customizes poster with text, font, layout
4. Creates account or logs in
5. Submits order and pays
6. Admin manually transforms image
7. Final image sent by email for user validation
8. Order status updated in user account

  1. Header: logo left, menu right (Accueil, Galerie, Personnaliser, Contact), responsive hamburger menu

  3. Product grid: 6 example posters, each with image, title, price, hover effect (zoom & shadow), responsive (3/2/1 columns)
  4. Section 'Comment ça marche': 3 steps illustrated with icons/placeholders, short text
  5. Footer: contact links, social media icons, legal info, simple design
- Use React functional components and TailwindCSS
- All images and icons can be placeholders (Unsplash / Heroicons)
- Make components modular, reusable, and responsive
- Include comments explaining each part
- Style guidelines:
  - Color palette: primary #BBD0CB, secondary #F9FAFB, accent #EF4444
  - Font: Inter, sans-serif
  - Buttons: hover scale 105%, transition duration 200ms
  - Product hover: scale 105%, shadow-md, transition duration 200ms
